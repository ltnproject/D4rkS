# D4rkzOS Website 🌐

Welcome to the official web repository for **[D4rkzOS](https://drk.qzz.io/D4rkS)**! 
This repository hosts the static landing page showcasing the features, aesthetic, and system requirements for the cyberpunk-themed Linux distribution based on KDE Neon.

## 🚀 Live Demo
You can view the live website here:
**👉 [drk.qzz.io/D4rkS](https://drk.qzz.io/D4rkS)**

## 🛠️ Built With
This landing page is designed to be blazingly fast and purely static:
* **Vanilla HTML5 & CSS3** for maximum performance and zero bloat.
* **CSS Grid & Flexbox** for a fully responsive, mobile-friendly layout.
* **No Frameworks** (No React, Vue, or heavy libraries).
* **Relative Pathing** specifically engineered to run gracefully on subdomains and GitHub Pages paths.

## 📂 Repository Structure
```
.
├── index.html                  # Main entrance & structure
├── .nojekyll                   # Bypasses Ruby rendering on GH Pages for instant deploy
└── assets/                     # Media & screenshots directory
    ├── desktop-screenshot.png
    ├── boot-logo.png
    └── wallpaper.png
```

## 🌐 Deploying
The codebase is naturally formatted for **GitHub Pages**. 
To deploy your own fork or mirror:
1. Push this folder to a GitHub repository.
2. Go to `Settings` -> `Pages`.
3. Select the `main` branch. 
4. The site will compile automatically! Thanks to the `.nojekyll` file, deployment is close to instantaneous.
