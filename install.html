Deployment Manual

How to Install D4rkzOS

A comprehensive walkthrough from initial acquisition to full cybernetic integration.

Phase 01

The Acquisition

Prepare your physical medium and acquire the latest system image.

  • 1

    Download the ISO

    Acquire the D4rkzOS-2.0-Ghost.iso (approx. 3.4GB) from our secure repository.

    Download v2.0 ISO
  • 2

    Flash to USB

    We recommend using Ventoy for multi-boot capability or Rufus (DD mode) for a dedicated drive. Ensure your drive has at least 8GB of capacity.

    Recommendation: Ventoy allows you to simply drag-and-drop the ISO without re-formatting.
Phase 02

Hardware Negotiation

Configure your BIOS/UEFI to accept the new operating environment.

  • 1

    Enter Firmware Settings

    Restart your machine and tap F2, F12, Del, or Esc to enter the BIOS/UEFI menu.

  • 2

    Critical Settings

    Ensure the following parameters are set:

    • Boot Mode: UEFI (Required for systemd-boot)
    • SATA Mode: AHCI
    • Secure Boot: Disabling is recommended for the smoothest experience, though modern kernels support it.
  • 3

    Initialize Live Session

    Save settings and boot from the USB. Select "Start D4rkzOS" from the boot menu. The Plymouth animation will greet you.

Phase 03

The Cybernetic Graft

Running the Calamares installer to commit D4rkzOS to your drive.

  • 1

    Launch Installer

    Click the Install D4rkzOS icon on the desktop or launch it from the Application Menu.

  • 2

    Partitioning Logic

    For a clean install, select Erase Disk. If you are a veteran, manual partitioning with an EFI partition mounted at /boot/efi is required.

    Data Loss: Erasing the disk will destroy all existing data. Back up your identity files first.
  • 3

    Verification & Execution

    Review the summary and click Install Now. The D4rkzOS branding will cycle through features as the system is deployed.

Phase 04

Calibration

Final touches to ensure your cyberdeck is peak performance.

  • 1

    First Login

    Remove the USB drive and reboot. Your customized Zsh terminal and Plasma desktop are now operational.

  • 2

    Synchronize Repositories

    Open the terminal (Ctrl+Alt+T) and run the primary sync command:

    sudo pkcon update
  • 3

    Explore the Suite

    Browse pre-installed tools like VS Code, btop, and Kdenlive. Welcome to the grid.
