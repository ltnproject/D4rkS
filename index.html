System Requirements

What You Need

D4rkzOS is optimized for modern hardware.

🟡 Minimum

CPUDual-core 64-bit (2015+)
RAM4 GB
Storage25 GB SSD/HDD
BootUEFI firmware
GPUAny with OpenGL 2.0
Recommended

🟢 Recommended

CPUQuad-core 64-bit (2019+)
RAM8 GB+
Storage50 GB NVMe SSD
BootUEFI + Secure Boot
GPUDedicated (AMD/Nvidia)
⚡

UEFI Required: D4rkzOS uses systemd-boot which requires UEFI firmware. Legacy BIOS systems are not supported in v1.0.
